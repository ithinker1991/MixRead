🧠 一、MixRead 的使命（一句话）

让用户不依赖全文翻译，而是通过“混合输入 + 难度可控”的方式，自然提升英文阅读能力。

这个使命决定了 MixRead 不只是“辅助阅读工具”，而是未来可以演化成一个完整的“阅读 + 学习 + 成长体系”。

🎯 二、核心价值（3 个价值点不可被替代）
① 阅读不被打断（核心爽点）

不同于全文翻译，MixRead 提供只针对“不熟悉词”进行即时解释的体验。

② 自动识别“你可能不认识的单词/短语”

这是 MixRead 的 AI 价值点。
用户不需要「主动查词」——系统自动标注用户可能需要关注的点。

③ 难度可调（混合阅读）

对中文页面：自动替换一些词/短语为英文

对英文页面：自动标注难词/短语

让用户逐步过渡到“原文能力”

这是 MixRead 的差异化能力，别人做不到。

🚀 三、MVP（2 周可以上线）功能清单

这是你的 最小可验证产品，目标是验证：

用户愿意连续使用 3 次，并觉得“爽”。

不要超过这些功能，否则你会做不完/走偏。

🧩 MVP 功能 1：英文文章的智能难词标注
功能细节：

Chrome 插件自动检测网页是否为英文

抽取文本

根据：

常见词频

词长

词级别（CEFR A1~C2）
自动标注“可能不认识”的词（下划线或浅色 highlight）

为什么是核心？

用户会立即感受到：“阅读压力被降低了”。

🧩 MVP 功能 2：悬停解释（英文解释为主，少量中文）

悬停时显示：

英文定义（简单版）

例句（从 LLM 或词典）

发音按钮（可选）

注意：
⚠️ 不要做中文翻译（会让用户依赖翻译）
只提供简明英文解释。

🧩 MVP 功能 3：难度调节（A0–C2）

一个滑杆：

低难度 → 标注更多词

高难度 → 标注减少

这是 MixRead 最核心的创新功能。

🧩 MVP 功能 4：点击加入词库（一个按钮就够）

用户可以把某个词加入“个人词库”。

你不需要做复习系统，只保存词库就够了。

🧩 MVP 功能 5：简单的“已学习词汇统计”

词库数量

每天新增词

每周阅读时长（可选）

让用户有成就感。

🎯 MVP 的最终目标（你需要验证）

用户是否连续使用 3 天？

是否愿意分享给朋友？

是否认为“阅读变轻松”？

只验证这三条。

🔥 四、可添加的功能（但绝对不是 MVP 阶段做）

以下是 MixRead 最终可以做到的，但 每一项都要放在 MVP 之后：

📚 可添加 1：中英文混合阅读模式（你方向的特色）

对于中文网站：

自动把 5% / 10% / 20% 的词替换成英文

用户可以调节比例

形成英文 exposure

这个功能非常亮眼，但不是 MVP。
为什么？

需要更复杂的 NLP

会引起排版问题

初期实现成本偏高
→ MVP 为了快速验证，先做英文页面增强即可。

👁️ 可添加 2：段落难度预测（用户阅读速度）

LLM 分析每段的难度并给出：

阅读速度

停留时间

词汇密度

难句次数

这些数据可以成为未来“阅读能力进阶”的基础。

🧠 可添加 3：AI 长难句拆解

类似：

对英文长句进行结构拆解（主干/从句等）

非常有价值，但不是 MVP。

📝 可添加 4：自动生成闪卡 + 复习系统

将词库变成可复习的内容。

适合第二阶段。

🗣️ 可添加 5：混合听力模式（语音增强）

利用你的流式语音经验：

背景朗读

生词单独发音

英文段落朗读

中/英交替阅读

未来 killer 功能，但不属于 MVP。

📱 可添加 6：手机端支持（WebView / Safari 插件）

第 3 阶段通过。

📈 五、产品分阶段路线图（非常重要）
🟦 第一阶段（0 → 第 1 个真实用户）

目标：验证“英文增强阅读”是否被真实用户喜欢

必须做：

难词标注

悬停解释

难度调节

词库（简单存储）

使用数据统计（简单版）

绝对不要做：

账户系统

支付系统

复习系统

混合中文功能

AI 长难句分析

移动端

目标：3 次有效使用。

🟩 第二阶段（10–1000 用户）

目标：增强“学习闭环”

可添加：

导航栏：阅读历史

词库 → 闪卡生成

每日复习

阅读报告/能力曲线

中文网页混合模式（5% 初版）

不要做：

AI 老师

零碎学习模块

全量课程内容

多设备同步复杂逻辑

目标：Retention > 20%。

🟧 第三阶段（1000–10 万用户）

目标：从工具 → 产品体系

可添加：

长难句分析

段落难度预测

内容推荐（RSS + AI）

移动端（React Native 或纯插件）

团队协作模式（可选）

支付体系（订阅制）

这是你的 商业化阶段。

🟨 第四阶段（产品成熟期）

目标：从阅读 → 完整语言能力平台

可能扩展：

口语练习

听力增强

写作纠错

AI 阅读指导

完整语言学习曲线

做到这里，MixRead 是一个 语言增强 OS。
