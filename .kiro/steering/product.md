# MixRead Product Overview

MixRead is an intelligent English reading enhancement tool that helps users improve their reading ability through word difficulty awareness and vocabulary learning.

## Core Features

- **Word Highlighting**: Automatically highlights difficult words on web pages based on CEFR difficulty levels (A1-C2)
- **Vocabulary Management**: Three-tier word tracking system:
  - Known words (user confirmed they know)
  - Unknown words (user marked as not knowing)
  - Vocabulary/Library (words user wants to learn with context)
- **Domain Management**: Blacklist/whitelist domains where the extension should be active
- **SRS Review System**: Spaced repetition flashcard system for vocabulary learning
- **Chinese Translations**: Support for Chinese translations of English words

## User Flow

1. User browses web pages with the Chrome extension active
2. Extension highlights words above user's difficulty threshold
3. User can mark words as known, unknown, or add to vocabulary
4. User reviews vocabulary using the SRS-based flashcard system
5. Mastered words are tracked and no longer highlighted

## Target Users

English learners who want to improve reading comprehension by identifying and learning unfamiliar vocabulary in context.
